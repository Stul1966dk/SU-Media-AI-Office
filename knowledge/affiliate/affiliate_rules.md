# Affiliate-regler

## Partner-ads

- Brug `kombiid` som unik salgsidentifikator.
- Gem rå og normaliserede salgsdata uden formaterede beløb.
- Send kun notifikation for nye salg.
- Log fejl uden at vise API-nøgler eller private feedadresser.
- Behandl provision som den primære forretnings-KPI.

## Produktartikler

- Beskriv, hvem produktet passer til, og hvem det ikke passer til.
- Underbyg fordele og ulemper med konkrete egenskaber.
- Opdatér pris, tilgængelighed, specifikationer og affiliatelinks.
- Brug tydelige CTA'er uden at love mere, end produktet kan levere.

## Kategoriartikler

- Hjælp brugeren med at forstå valgets vigtigste kriterier.
- Sammenlign relevante produkttyper og forklar forskellene.
- Link til uddybende produktartikler og relevante brandsider.
- Prioritér kategorier med dokumenteret efterspørgsel og indtjeningspotentiale.

## Brandsider

- Forklar brandets position, sortiment, styrker og begrænsninger.
- Undgå generiske beskrivelser, der kan passe på alle brands.
- Link til relevante kategorier og produkter.
- Hold oplysninger om sortiment og forhandlere opdaterede.
